# absesnsi

Halaman absensi wajah modern dibuat dengan HTML, CSS, dan JavaScript murni.

File utama:
- `index.html`
- `styles.css`
- `script.js`

Untuk menyimpan data ke Google Spreadsheet, isi `GOOGLE_SHEET_URL` pada `script.js` dengan URL Web App Google Apps Script atau endpoint Google Forms yang digunakan untuk menambahkan baris.
